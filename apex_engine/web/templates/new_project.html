Seed Composition Guide

Understanding Latent Diffusion

Sonauto uses a latent diffusion model, not a token-based model like Suno. This means control is achieved through semantic embeddings and parametric steering rather than bracketed commands.

The model "hears" the global context holistically. It interprets your prompt as an aggregate semantic vector that guides the denoising process from noise to audio.

Prompt / Description Field

This is where you define the texture and attitude of the track. Focus on:

  • Instrumentation: "heavy 808 bass", "rolling hi-hats", "sampled jazz piano"
  • Atmosphere: "dark", "dystopian", "nocturnal", "triumphant"
  • Vocal Delivery: "aggressive flow", "melodic hook", "staccato delivery"
  • Production Quality: "high-fidelity", "studio-quality", "wide stereo imaging"

Lyrics Field

Structure your lyrics with meta-tags that the model recognizes:

  • [Intro] - Usually instrumental or spoken ad-libs
  • [Verse] / [Verse 1] - Main lyrical content
  • [Chorus] - The hook, often more melodic
  • [Bridge] - Transitional section
  • [Outro] - Fading out or final ad-libs

Formatting tips:

  • Use line breaks to indicate pauses (since (breath) doesn't work)
  • Use CAPS to signal intensity shifts
  • Use ellipses ... for dramatic pauses
  • Parenthetical text (yeah!) may be interpreted as ad-libs

Neuropsychological Effects

Target specific emotional responses:

  • Frisson (chills): Triggered by dynamic surges, spectral expansion, expectation violation
  • Tension/Release: Build with sparse arrangement, release with full instrumentation
  • Emotional Arc: Plan how the track should make listeners feel over time

Neurochemical Targets

Engineer dopaminergic engagement through rhythm:

  • Syncopation: Off-beat emphasis creates prediction error (dopamine release)
  • Groove Quality: Micro-timing variations make beats feel "human"
  • Earworm Potential: Repetition + novelty for memory encoding

Musical Effects

Production techniques and mixing notes:

  • Tempo Feel: Half-time, double-time, triplet hi-hats
  • Processing: Sidechain compression, reverb, delay
  • References: Name specific artists or production styles

CFG Scale (Prompt Strength)

This controls how strictly the model follows your prompt:

  • 1.5-2.0: Natural genres (pop rap, melodic, acoustic, jazz rap)
  • 2.0-2.5: Balanced genres (trap, hip hop, boom bap, conscious rap)
  • 2.5-4.0: Aggressive genres (phonk, drill, industrial) - artifacts become aesthetic
